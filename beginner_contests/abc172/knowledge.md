# 累積和
- 配列の区間内の和を求められる際に、事前計算をしておくことで計算量を減らす
```
[a0, a1, a2, a3, a4]の時に
s1 = a0
s2 = a0 + a1
s3 = a0 + a1 + a2
...
をsとすると
s5 - s2をするだけで、O(1)で3~5の区間の累積和を得られる
```
